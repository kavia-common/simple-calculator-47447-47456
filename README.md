# simple-calculator-47447-47456

This project contains a Svelte-based simple calculator frontend (calculator_frontend) themed "Ocean Professional". It features basic arithmetic, percent, decimal input, clear (C), backspace (CE), toggle sign (±), sequential operations with immediate execution, continuous equals, and error handling. Keyboard shortcuts: digits 0–9, ., + - * /, %, Enter, Backspace, Escape.

To run the frontend:
1) cd calculator_frontend
2) npm install
3) npm run dev (runs on port 3000)
